The following option launches the selected Development Tool from the left navigation bar.

The Minecraft-DT2 launcher will shutdown after successful execution of the selected Development Tool.

The following option checks if a new version of the Minecraft-DT2 launcher is available. All release notes are then displayed in the main body.

Note that this option requires an internet connection to compare the necessary data from our official website API.
